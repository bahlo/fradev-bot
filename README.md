# fradev-bot

The bot of the Frankfurt Developer Slack.

## Installation

With node installed, run the following:

```javascript
$ npm install
$ SLACK_TOKEN=abcdef123456789 npm start
```

## Current features

- New users get greetet and asked to put in a first and last name if not
  already set.
